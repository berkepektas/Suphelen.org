# SuphelenOrg

**SuphelenOrg**, AFAD ve diğer resmi kurum sitelerinden alınan haber ve duyuru linklerini doğrulamak ve güvenilir bilgi sağlamak için geliştirilmiş bir projedir. 
Amacı, dezenformasyon ve yanlış haberlerin önüne geçmek, kullanıcıların güvenilir bilgiye hızlı bir şekilde ulaşmasını sağlamaktır.

## Özellikler
- AFAD ve diğer resmi kaynaklardan veri çekme
- Linklerin doğruluk ve güvenilirlik kontrolü
- JSON veya tablo formatında raporlama
- Basit ve anlaşılır kullanıcı arayüzü (terminal veya web tabanlı)

## Kurulum
1. Projeyi klonlayın:  
git clone <repo_link>
Python 3 yüklü olmalı

Gerekli kütüphaneleri kurun:
pip install -r requirements.txt

2.Kullanım
Terminal üzerinden çalıştırın:
python main.py
İlgili URL’leri girin ve doğruluk raporlarını alın
Çekilen haberler JSON dosyası olarak kaydedilebilir

3.Katkıda Bulunma
Herkes katkıda bulunabilir, pull request gönderilebilir
Yeni doğrulama metodları ekleyerek projeyi geliştirebilirsiniz
