# SuphelenOrg

Bu proje, AFAD ve diğer resmi kurum sitelerinden alınan haber linklerini doğrulamak için geliştirilmiştir.

## Özellikler
- Dezenformasyon tespiti
- AFAD resmi verileri ile karşılaştırma
- Doğruluk payını hesaplama

## Kullanım
1. Projeyi klonlayın: `git clone <repo_link>`
2. Python ile çalıştırın: `python main.py`
